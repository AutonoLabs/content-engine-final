# Content Type Taxonomy — 16 Formats

> All content types the engine supports. Rotate across posts to maintain diversity. Updated 2026-07-08.

---

## Why this doc exists

Posts of all one type get stale fast. Algorithm fatigue is real:
- Instagram: feed variety boosts reach (mix static + carousel + video)
- LinkedIn: posts that mix docs + text + video outperform text-only streams
- X: threads + one-liners + media posts each perform differently, rotation matters
- TikTok: video only, but you can rotate tutorial-style vs. story-style vs. trend-style

Beyond algorithm, **variety serves the audience**: they stay because they don't know what to expect.

## 16 content types

### Static formats

| Type | Description | Best platforms |
|---|---|---|
| **Static image — square** | 1:1 single image post | IG, LinkedIn, X, FB |
| **Static image — landscape** | 16:9 single image post | LinkedIn, X, FB |
| **Static image — portrait** | 4:5 single image post | IG, FB |
| **Carousel** | multi-image swipe (3-10 slides) | LinkedIn, IG |
| **Quote graphic** | image of a quote, designed | LinkedIn, IG, X |
| **Poll/question graphic** | image with poll/question overlay | IG stories, LinkedIn, X |

### Text-only formats

| Type | Description | Best platforms |
|---|---|---|
| **Text-only post** | pure written post, no media | LinkedIn, X, Threads, FB |
| **Thread** | multi-post series (3-10 connected posts) | X, Threads, LinkedIn |
| **Question post** | text that asks, designed for engagement | LinkedIn, X, Threads |

### Long-form formats

| Type | Description | Best platforms |
|---|---|---|
| **PDF article** | downloadable PDF, linked in post | LinkedIn (document post), X (link), Substack |
| **Presentation** | slide deck, often shared as document | LinkedIn (document post), SlideShare |
| **Long-form article** | 1000+ words, blog/newsletter | LinkedIn (link), X (link), site |

### Video formats

| Type | Description | Best platforms |
|---|---|---|
| **Short video — vertical** | 6-90s vertical, native platform upload | TikTok, Reels, YT Shorts |
| **Long-form video** | 2-30min horizontal | YT, LinkedIn, FB |
| **Live / upcoming** | event, launch teaser, behind-the-scenes live | IG Live, LinkedIn Live, Twitch |

### Engagement formats

| Type | Description | Best platforms |
|---|---|---|
| **Behind-the-scenes** | process reveal, workspace | IG stories, TikTok, Threads |
| **Review (own product)** | reviewing your own product/feature | IG, TikTok, YT |
| **Review (third party)** | reviewing another product/brand | all platforms |
| **Comparison** | us vs them, or product A vs B | LinkedIn, X, IG, TikTok |
| **Link share** | article + commentary | LinkedIn, X, Threads, FB |

## When to use each

Decision rules (defaults — adjust per brand):

- **B2B brands:** lead with text-only + PDF + LinkedIn carousel. Video is secondary.
- **Consumer brands:** lead with video + visual image. Text-only is rare.
- **Service / professional:** lead with text-only + PDF + comparison. Visual is secondary.
- **Education / knowledge:** lead with carousel (IG, LinkedIn) + video (YT, TikTok).
- **Story / advocacy brands:** lead with video + behind-the-scenes.

---

## Mixing across posts

Target distribution per week (varies by brand):
- Image: 25-30%
- Text-only: 20-25%
- Video (short): 25-35%
- Carousel / PDF / long-form: 10-15%
- Engagement formats (review, comparison, BTS): 10-15%

Track in `content-mix.md` per brand.
