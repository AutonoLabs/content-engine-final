# Target Audience Template

> Per-brand override layer. Sits between platform demographics and voice profile. Updated 2026-07-07.

---

## Why this template exists

Platform demographics (`docs/audience-demographics.md`) describe the **baseline user** on each platform. But most brands aren't trying to reach the average user — they're trying to reach a specific subset.

Examples:
- AllSquared on LinkedIn: platform baseline is 25-34, B2B professionals. But AllSquared targets **UK trades + agencies + producers + clients** specifically — that's a sub-slice within the B2B baseline.
- Yapper on TikTok: platform baseline is 18-34, consumer discovery. But Yapper targets **young women managing chronic health conditions** — a specific sub-slice with different emotional register.
- Politicall on X: platform baseline is news-driven, 25-34, 64% male. But Politicall targets **US political campaign managers + operatives** — high-context, insider register, B2B-ish despite consumer surface.

Without this layer, the agent defaults to platform-baseline assumptions and misses the actual target.

## Tone calibration rules

When platform baseline ≠ brand target, the voice adjusts:

| Gap | Voice adjustment |
|---|---|
| Brand target is **narrower** than platform baseline | Get specific. Use industry shorthand. Don't explain basics. Talk like an insider. |
| Brand target is **older** than platform baseline | Slow down. Less trend-chasing. More authority. More context. |
| Brand target is **younger** than platform baseline | Faster cuts. More native slang. Less explanation. Visual-first. |
| Brand target is **higher-income** than platform baseline | Premium register. Quality > quantity. No budget talk. |
| Brand target is **lower-income** than platform baseline | Accessibility. Plain English. Value framing. |
| Brand target is **more B2B** than platform baseline | Business register. ROI logic. Regulatory clarity. |
| Brand target is **more consumer** than platform baseline | Lifestyle language. Emotional hooks. Visual storytelling. |
| Brand target is **more female** than platform baseline | Listen-first. Visual quality high. Avoid bro-isms. |
| Brand target is **more male** than platform baseline | Direct. Numbers-heavy. Skip soft openers. |

## Example: AllSquared (current state)

This is what got filled in for AllSquared. Use as reference when filling in for other brands.

### Target audience

**Primary:** UK construction project owners + UK agencies/producers doing £20k-£200k projects. Geographic focus: London + SE + Midlands. Pain: trust deficit after contractor scam or platform dispute gone wrong.

**Secondary:** UK builders/trades doing £50k+ projects who want milestone-based payment + dispute protection.

### Platform-by-platform

- **LinkedIn:** Primary audience = UK construction project owners + agency owners. Tone = founder-to-founder, regulatory clarity. Ignore generic B2B marketing.
- **X:** Primary = UK trades Twitter + property journalists. Tone = punchy, contrarian, news-anchored. Ignore crypto/web3.
- **Instagram:** Primary = UK homeowners researching projects. Tone = visual storytelling. Ignore pure B2B decision-makers.
- **TikTok:** Primary = trade apprentices + DIY-curious. Tone = casual native, fast. Ignore pure entertainment.
- **YouTube:** Primary = trade professionals learning + homeowners researching. Tone = authoritative + accessible. Ignore pure entertainment.
- **Threads:** Primary = UK construction Twitter overflow. Tone = casual industry chat. Ignore productivity Twitter.
- **Facebook:** Primary = older homeowners 45+ + local community groups. Tone = accessible, traditional. Ignore B2B.
