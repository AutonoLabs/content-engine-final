# Examples

The worked example used to live here as a copy of the AllSquared brand folder,
which drifted from the live one. It's gone — **the live brand folders are the
reference**:

- `brands/allsquared/` — fintech brand, full weekly batches (W29, W30),
  media packets, content-mix + performance-log in canonical format
- `brands/yapper/` — consumer-health brand with Higgsfield workflow docs

For the file formats, see:
- `docs/post-file-spec.md` — post files with frontmatter (machine state)
- `brands/*.template.md` — the 7 brand templates with inline examples
